SimpleUDPMulticastChat
======================

Simple Lan Party UDP Multicast Chat Client with GUI based on Java &amp; Swing.

Why use this @ your next LAN-Party?
===================================
*  no server needed
* fast communication (i.e. sharing server IP's)
* it's old school!
